Bootcamp 2020: Caesar Cipher Encryption Program

My first programming project, concluding the 2 week Programming Bootcamp at City (2020). This Processing project includes the use of user interaction, loops and methods. This encryption program shifts the characters the user inputs by the number the user chooses. Since there are only 26 letters in the alphabet, the use of a for loop means if the shift goes passed the letter 'Z', it will loop back round to 'A' and continue. I also have a type writing effect when the user inputs their password and detects the number they input for the shift. Furthermore, I have also added a button which detects when the mouse is pressed inside which will run the Cipher method and encrypt the users input.
